🛠️ STEP 1 - INSTALL FABRIC

Fabric is the thing that lets all the mods work.

Open the Fabric installer.
Select Minecraft 1.26.2.
Click Install.
Done.

Without Fabric the game will crash.

📦 STEP 2 - INSTALL THE MODPACK
Open Minecraft folder.

Press:

Windows Key + R

Type:

%appdata%

Press Enter.

Open:

.minecraft
Find the folder called:
mods

If it doesn't exist, create it.

Copy ALL mod files from the modpack into:
.minecraft/mods

Done.

⚡ PERFORMANCE MODS

"Why does this modpack have optimization mods?"

Because Minecraft is dumb.

These mods make the game run smoother.

Sodium

Makes Minecraft use your graphics card better.

More FPS.

Less lag.

Lithium

Makes Minecraft think smarter.

Improves game performance without changing gameplay.

Entity Culling

If you can't see something behind a wall...

Minecraft won't waste power rendering it.

More FPS.

BadOptimizations

Fixes lots of small performance problems.

Free FPS.

ServerCore

Makes servers run better.

Less lag.

Better TPS.

Spark

Lets us check what's causing lag if something breaks.

Basically a doctor for Minecraft.
